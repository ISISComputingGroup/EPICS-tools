Detailed Description

Each sub-class of this class defines a Substitution.

A Substitution is defined by specifying at least the following class members in the subclass:

Parameters:
Arguments List of names of Substitution arguments that must be filled in when specifying a Substitution.
TemplateFile Name of template file to be loaded. By default this will be looked for in the db subdirectory of the library.
SubstitutionOverwrites Where definitions from one template override an earlier template it is necessary to ensure that templates are loaded in a particular sequence. This is done by setting this varible to a list of all templates which need to be loaded before this one.
