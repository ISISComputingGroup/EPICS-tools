def iocbuilder.fanout.create_dfanout (   name,
  record_list,
  args

Creates one or more data fanout records (as necessary) to fan a data output out to a list of records.

If no more than 8 records are given then this creates a single dfanout record called name. Otherwise, further records are created and daisy-chained together: subsequent records are numbered in sequence.

Parameters:
name Name of first fanout record. Subsequent records have a sequence number appended to this name.
*record_list List of records to be processed.
**args Extra field definitions to be passed to the generated fanout records.
