00094     # \param record

00097     #   normally the device part of the record name is not specified here.
00098     # \param **fields

00100     #   of the class.  Values can be specified in the constructor, or can be
00101     #   assigned subsequently to the generated instance.

00103     # For example, the following code generates a record which counts how
00104     # many times it has been processed:
00105     #

00107     #   cntr = records.calc('CNTR', CALC = 'A+1', VAL = 0)
