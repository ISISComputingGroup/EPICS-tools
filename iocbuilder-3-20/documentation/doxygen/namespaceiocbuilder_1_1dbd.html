Implements the set of records provided by a dbd

Variable Documentation

iocbuilder::dbd.records = RecordTypes

Every record type loaded from a DBD is present as an attribute of this class with the name of the record type.

For example, to create an ai record, simply write 

   records.ai('NAME', DESC = 'A test ai record', EGU = 'V')

Each record is of type Record, see for more details.
