Detailed Description

This function produces an ArgInfo list for the xml frontend to iocbuilder.

It uses the arguments and defaults from the supplied init method and annotates them with types and descriptions. Example:

 class mybase(ModuleBase):
     def __init__(self, arg1, arg2=1):
         self.__super.__init__(self)
         ... do other stuff...

     ArgInfo = makeArgInfo(__init__,
         arg1 = Simple('Description for arg1', str),
         arg2 = Simple('Description for arg2', int))
